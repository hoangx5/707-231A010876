NỘI DUNG LÝ THUYẾT

1. position: absolute và relative

relative: định vị tương đối so với chính nó, vẫn chiếm chỗ.

absolute: định vị tuyệt đối so với phần tử cha gần nhất có position.

Ví dụ:

.box { position: relative; top:20px; } .child { position: absolute; top:0; right:0; }

Responsive: relative dùng cho layout chính, absolute dùng cho icon, overlay.

Deploy: nếu dùng absolute nhiều dễ vỡ layout trên mobile.

2. Domain, Hosting và DNS

Quy trình:

  • Đăng ký domain
  • Mua hosting
  • Trỏ DNS

DNS: chuyển domain → IP server.

Công cụ deploy:

  • GitHub Pages
  • Netlify

Rủi ro:

  • Bị hack DNS
  • Không có HTTPS
  • Lộ API key

3. let, const, var

var: function scope, dễ lỗi

let: block scope

const: không đổi giá trị

Ví dụ lỗi:

for (var i=0;i<3;i++){ setTimeout(()=>console.log(i),1000); }

Kết quả: 3 3 3 (sai)

Fix:

for (let i=0;i<3;i++){ setTimeout(()=>console.log(i),1000); }

Debug: dùng Chrome DevTools (F12)

4. Event Delegation

Gắn event vào phần tử cha thay vì từng phần tử con.

Ví dụ:

document.getElementById("list").addEventListener("click", function(e){ if(e.target.tagName==="LI"){ alert(e.target.innerText); } });

Lợi ích:

  • Tăng hiệu suất
  • Hỗ trợ phần tử động
  • Code gọn
